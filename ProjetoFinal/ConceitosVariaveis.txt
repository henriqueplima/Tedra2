	Para resolver problemas no computador é necessário manipular dados. Cada variável corresponde a uma posição da memória na qual seu conteúdo (valor) pode ser alterado durante a execução do programa. É por isso que o nome desse conceito é variável, pelo fato de podermos alterar o seu conteúdo (valor).#	Imagine uma mesa contendo caixas em cima dela, onde cada caixa contém pastas com dados de funcionários e uma etiqueta de identificação. Neste exemplo a mesa representa nosso espaço de memória e as caixas representam as variáveis. Para que algum funcionário pegue dados de alguém, precisa ir até a mesa (memória) escolher a caixa(variável) e acessar os dados necessários. Depois de utilizar esses dados o funcionário volta para a mesa e devolve os dados já alterados na caixa.#Tipos de dados	Cada tipo de variável necessita de um espaço diferente em nossa memória sendo que não se pode guardar informações  em uma variável que não seja do mesmo tipo do dado.Existem 3 principais tipos de variáveis:#Numéricas:1) Inteiro: são representados por números inteiros exemplo: -1, 100, 9999.2) Real: são representados por números fracionados exemplo: 0.5, 1.4, 10.9.Obs: a casa decimal deve ser separada por ponto “.“ ao invés de vírgula.#Caracteres:1) Caractere: é representado por apenas um caractere, podendo ser letras do alfabeto, caracteres especiais ou até mesmo números, exemplo: h; j; k; a; %; &; 1; 4.obs. Caso seja salvo um número como caractere, não será possível efetuar nenhuma operação matemática com ele.2) String: é representado por uma cadeia(conjunto) de caracteres, muito utilizado para manipular nomes, frases e etc. Exemplo : computador; escola; professor; 95678-0034.obs. Caso seja salvo um número como string, não será possível efetuar nenhuma operação matemática com ele.#Lógicas(Booleanas):	Variáveis lógicas ou booleanas armazenam somete duas informações diferentes, verdadeiro ou falso. 	Imagine que dentre as várias informações referente a um funcionário uma delas seja para saber se o mesmo está de férias. Quando o funcionário vai até a mesa(memória) e acessa a caixa (variável) “Funcionário de férias“, ele encontre um valor verdadeiro(sim) ou falso(não).#Declarando Variáveis	Para criar uma variável que seja reconhecida pelo sistema e necessário declara-la, dando um nome e um tipo para ela.  As variáveis são identificadas pelo nome que é dado pelo próprio programador.	Imagine que em nossa mesa possua diversos espaços e queremos colocar novas informações lá, para isso precisamos colocar uma nova caixa, lembrando que a caixa deve ter uma etiqueta com o nome e um tipo. Após termos reservado essa caixa já podemos guardar dados lá.#	A declaração de variáveis segue o seguinte padrão : <tipo> <nome>.Veja abaixo um exemplo de como isso é feito.string funcionario;real salario;	Neste exemplo declaramos duas variáveis uma do tipo “string“ para armazenar o nome do funcionário e outra do tipo “real“ para armazenar o salario.#	Obs. As variáveis diferenciam letras maiúsculas das minúsculas ou seja ”nome” é diferente de ”Nome” que por sua vez é diferente de ”NOME”. Este padrão é seguido na maioria das linguagens de programação. Outra coisa importante é que não podemos declarar uma variável com um nome que comece com um numero, exemplo:inteiro 1funcionario.Inteiro 12345.
